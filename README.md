# 🧮 Python Calculator

A simple GUI Calculator built using **Python** and **Tkinter**.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear (AC)
- Simple and user-friendly interface

## Technologies Used

- Python 3
- Tkinter

## How to Run

1. Clone this repository:

```bash
git clone git@github.com:Tushar-Barman/calculator.git
```

2. Open the project folder.

3. Run:

```bash
python calculator.py
```
